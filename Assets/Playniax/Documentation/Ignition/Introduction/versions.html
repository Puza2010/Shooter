Version 3.0

  • We have completely redone the particle effects, now using the Unity particle system instead of the deprecated Playniax particle system.
  • Texture2DHelpers class introduces a new method, Blit, which facilitates overlaying one texture onto another while employing alpha blending.
  • Texture2DHelpers class now includes a new method named GetScreenshot(), designed to capture and retrieve a screenshot of the current display.
  • Folder restructuring or renaming.
  • Added tons of tooltips to the classes.
  • Improved documentation.
