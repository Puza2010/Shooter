Class EasyGameUI

Description

The EasyGameUI will manage the different screens and can hold game data that has to be maintained throughout the game session like number of lives, current level, purchases, etc.

Public fieldsDescription
Mode modeOperation Mode.
string backgroundSceneName = "Background Scene"Background scene to load.
PlayerSettings playerSettingsThe number of lives, etc.
PreLevelSettings preLevelSettingsThe scene(s) containing the intro(s).
PostLevelSettings postLevelSettingsThe transition scene will be shown after each level is completed.
AdvertismentSettings advertisementSettingsThe scene(s) containing the ad(s).
bool showLoadingProgress = trueWhether the loading progress screen is showed during loading.
Font fontFallback font.

Some examples of how to use the public fields or methods:

Get the totatl number of lives: 

Debug.Log(EasyGameUI.instance.playerSettings.lives)
The lives counter during the game: 

Debug.Log(PlayerData.Get(0).lives)
Where the zero indicates the player's index (0 = actually player 1, 1 = player 2, and so forth)

Load the next level: 

EasyGameUI.instance.Levelup()
Stop the game and show the Game Over page: 

EasyGameUI.instance.GameOver()
